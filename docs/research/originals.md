## 1. slide-builder（スライドプロンプトジェネレーター）

### フォーム構造

- 03 デザイン（必須）: 枚数1〜10（ChatGPT Images 2.0 は1回10枚まで、11枚以上は章分割）/ デッキタイプ（単独・最初の章・中間章・最終章 → 役割ラベルが章用に切替+「全○章中の第×章」文脈）/ ビジュアルスタイル約45種（コーポレート・クリーン、戦略コンサル、ビビッド各色フラット、2色フラット多数、アイソメ4種、SaaS、金融・伝統、法律・士業、行政・公共、採用・会社案内、ヘルスケア、クリニック、メンタルケア、ミニマル・モノトーン、雑誌風エディトリアル、ポートフォリオ、ブック、報道レッド、ウォーム、アカデミック、キッズ、和・伝統モダン、アパレル、グルメ、建築、クラフト、ラグジュアリー、ジュエリー、上質パステル、エレガント、ボタニカル、ポップ、クリエイティブ広告、ジム、キャラ解説、マンガ風、ゆるキャラ、絵本風水彩）/ 書体（自動切替+上書き: ゴシックモダン・極太・明朝・手書き風）/ 視認性（高・標準・編集的小さめ）
- 04 各スライドの内容（任意）: スライドごとに役割ピル（表紙/イントロ/本編/まとめ・次行動/なし）+ 内容1行
- 05 詳細設定: ブランドカラー / ページ番号 / フッターにタイトル / 表紙メッセージ位置（左右）/ まとめレイアウト（左右分割・上下二段）/ クレジット（作成者・所属・日付・連絡先・CTA文言）/ 追加指示

## 3. note-diagram-builder（note図解プロンプトジェネレーター）

### フォーム構造
- STEP 00: 記事本文ペースト（大textarea）。`[図解]` マーカーと `##` 見出しを図解候補として検出、`[図解：テーマ]` でテーマ指定可。カウンタ表示（n CHARS / n [図解] MARKERS / n HEADINGS）。本文はプロンプト末尾に【参考：記事本文】として同梱
- 01 デザイントーン: 表面（フラット/アイソメ/ソフト3D）× パレット17種（ネイビー×コーラル等の2色系・ビビッド1色系・モノ+1色）または完結系スタイル20種（キャラ解説・動物キャラ・マンガ風・棒人間・ピクトグラム・ペン画・水彩・クレイ3D・コーポレート写真調等、表面トグル無効）。選択中スタイルの説明文をライブ表示
- 02 仕様: 比率（16:9 / 4:3 / 1:1）/ 情報密度5段階（極ミニマル〜読み物寄り）/ 見出し・サブコピー（おまかせ・あり・サブのみ・なし）/ キャプション / 背景色 / 枠線
- 03 各図解の指定: 枚数1〜10ステッパー + 画像ごとに役割（自動・サムネイル・図解挿絵・まとめ）・見出し上書き・図解タイプ（おまかせ・フロー図・比較表・概念図・データ図解・比喩イラスト・チェックリスト）
- 04 詳細: ブランドカラー / トーンの引き継ぎ（回答分割時）/ 登場キャラに添付画像を使う / 追加指示
- ローカルストレージ自動保存を明記

### 生成プロンプトの特徴（実物確認済み）
- 「あなたはプロの編集デザイナーです」で開始
- 【コンセプト】で「本文の流れを邪魔しない編集的トーン」、テーマは末尾の記事本文から読み取らせる
- 【ビジュアルスタイル】は HEX 基準色つき（例: #F2863C を厳密に守る）
- [デザインルール] に「本文との馴染み」「下部キャプション（センタリング・罫線なし・余白でゾーニング）」「背景白統一」「枠線なし」
- [禁止事項] に「本文転載禁止（要点だけ図解化）」が追加
- 役割指針: サムネイル=強め1枚 / 図解・挿絵=静かな図版 / まとめ=CTA風装飾は避けて読了の余韻

## 4. ghost-writer（Writing Tone Builder）

記事ライティングのトーン&ルールMD（スタイルガイド）を生成。プロファイル保存・共有リンクあり。

### フォーム構造
- モード: 新規記事作成 / トーン調整md（既存文の調整用）
- 執筆媒体: note / WordPress / そのほか
- この記事について: SEO ON/OFF → キーワード最大3 + タイトル
- 構成: AIにお任せ / 手動入力。テンプレ8種（指定なし・まず結論から・要点→詳しく→まとめ・困りごと解決・体験談から学び・学習ステップ・おすすめランキング・用語しくみ解説）/ 導入に見出しを含めないトグル / 各見出しの内容を補足
- 読者: 想定読者 / よくある悩み（1行1つ）/ ゴール（1行1つ）
- 声・文体: 一人称（私・僕・筆者・使わない）/ 二人称（あなた・みなさん・使わない）/ 文体（ですます・である）/ 柔らかさスライダー
- トーン: 共感（しっかり寄り添う・控えめ）/ 煽り言葉（控えめ・禁止・要所で）/ 絵文字（禁止・あり）
- 表現ルール: ！の使用（禁止・控えめ・普通・多め）/ 禁止語カスタム（1行1語）
- 業種別法令ルール（multi・ONで禁止語と書き方ルールをMD出力に自動追加）: 健康食品サプリ（薬機法・健康増進法・景表法）/ 化粧品美容（薬機法56効能・景表法）/ 医療クリニック（医療法・医療広告GL）/ 不動産（宅建業法・公正競争規約）/ 金融投資保険（金商法・保険業法）/ 教育スクール（景表法・特商法）/ 景表法共通（優良誤認・有利誤認・二重価格）
- Markdownルール: 見出しH2/H3・箇条書き・強調・改行
- プレビューは目次ツリー + 本文。出力は「あなたはプロの編集者/ライターです」+ 条件列挙型MD

## 5. image-md（Nanobanana / 汎用1枚画像のMDプロンプト）

### フォーム構造
- 利用用途: サムネイル / 1枚図解 / 漫画（用途で表示項目とおすすめが切替）
- 比率9種: 1:1 / 4:3 / 3:4 / 16:9 / 5:2(X記事) / 9:16 / 3:2 / 2:3 / 21:9（それぞれ用途注記つき）
- サムネ設定: サブコピー（表示形式: そのまま・吹き出し・ボックス・角丸ボックス）/ キャッチコピー3行（1行8〜14字目安・未入力行は無視）/ レイアウト（ダイナミック中央・ダイナミック左上・斜め左上・高級繊細中央・高級繊細左上）/ バッジ表示トグル / ボタン表示トグル
- 画風11種: ミニマル線画 / コーポレート・フラット図解（万能）/ アイソメ / ピクト・アイコン / 3Dパステル / ゆるいイラスト / 手書き風 / レトロ印刷リソグラフ / ペーパークラフト / セル塗りアニメ / 指定なし（添付画像参照）
- 添付画像を参考に生成トグル
- 背景7種: 白ベタ推奨 / アクセント薄・濃 / グラデ薄・濃（背景のみ・ノイズでバンディング対策）/ 方眼紙 / ドット
- 配色: ルール（白黒+1色推奨 / モノクロ / 2色上級者）/ アクセント20色スウォッチ+自由入力 / アクセントの使い方（強調マーカー・見出し重要語・バッジ帯・アイコン・枠矢印・面・数字グラフ、1役割に絞る）

- 登場人物: 0〜3人 + 人物ごと（性別・服装 フォーマル/カジュアル/自由・添付画像忠実再現トグル・補足）
- その他: アイコン禁止トグル / AIへの追加指示
- 機能: リセット / コピー / MDダウンロード / 設定JSON書き出し・読み込み
