# ファーストビュー（FV/hero）パターン集

FV専用のAI画像生成プロンプトを組み立てるツールのマスターデータ。機械可読版は同ディレクトリの `firstview.json`。

- 前提: ユーザーは最初の約3秒で読むか離脱するかを決める（LPの直帰率は70%超とも）。FVの仕事は「読む理由」を3秒で渡すこと
- 視線はZ型/F型。左上＝誰の何のサイトか、右下＝次の行動（CTA）が基本配置
- 1つのFVに主役は1つ（キャッチ／ビジュアル／CTAのどれか）。先に決めてから配置する

---

## 1. FVレイアウトパターン（16種）

各パターンの `prompt`（配置記述文）は firstview.json に日本語・英語の両方を収録。以下は要約。

|---|----|------|-----------|---------|-----------|

| 2 | `split-text-right` | 右テキスト×左ビジュアル | エモーショナル・上質 | 美容、ブライダル、ジム | 左55%写真（被写体は右向き）、右45%テキスト。CTAが視線終点の右下に来る |

| 13 | `magazine-editorial` | 雑誌風エディトリアル | ハイセンス・上質 | アパレル、コスメ、メディア | 右2/3裁ち落とし写真、左1/3余白に細身セリフの段違いコピー＋号数風飾りテキスト |

### プロンプト合成の使い方

```
layouts[].prompt（配置） + 被写体の具体（業種の人物・商品・店内）
+ トーン指定（配色・質感・書体の雰囲気） + 枠指定（"Webサイトのファーストビューのデザインカンプ"）
```

- テキストは崩れやすい → 「コピーはプレースホルダーのダミーテキストとして」と添えるか、生成後に載せ替える前提
- アスペクト比は必ず明示（PC: 16:9 / 3:2、SP: 9:16）
- ネガティブ指定の定番: 文字の破綻、要素の重なりすぎ、ウォーターマーク

## 2. FV要素チェックリスト

### 大原則：「誰の・何の店で・どこにあって・次に何をすべきか」

FVを3秒見ただけで次の4問に答えられなければ失格。

1. **誰の**（どの会社・店の）サイトか → ロゴ・屋号
2. **何の店・何のサービス**か → メインビジュアル＋キャッチコピー
3. **どこにある**か（商圏があるなら）→ 地域名・駅名・アクセス
4. **次に何をすべき**か → CTA

判定法＝3秒テスト：初見の人に3秒だけ見せて閉じ、4問に口頭で答えてもらう。

### チェックリスト

| 要素 | 優先度 | ポイント |
|------|--------|---------|
| キャッチコピー | 必須 | 10〜25文字・最大2行。FVで最も大きい文字 |
| サブコピー | 推奨 | キャッチが情緒なら具体を、具体なら補強を。役割を重複させない |
| CTAボタン（主） | 必須 | FVで最も彩度の高い色。動詞で書く。1つに絞る |
| CTAボタン（副） | 任意 | 「詳しく見る」等の逃げ道。主CTAより明確に弱く |
| グローバルナビ | コーポレート必須 | ロゴ＋5±2項目＋問い合わせ。LPでは消す/減らす |
| ロゴ | 必須 | 左上が定位置。LPでも残す |
| メインビジュアル | 必須 | 画像だけで「何の店か」が分かるか。フリー素材感NG |
| 実績・受賞・掲載バッジ | 推奨 | 3つ前後。数字は具体（1,200社 > 多数） |
| 導入企業ロゴ | BtoB推奨 | 5〜7社、グレースケールでFV下端 |
| 電話番号 | 店舗系必須 | 受付時間とセット。SPはタップ発信 |
| 営業時間・定休日 | 店舗系必須 | 電話・住所と1ブロックに |
| 所在地・アクセス | 店舗系必須 | 駅名・徒歩分数。キャッチに織り込むとSEOにも効く |
| お知らせ・キャンペーン帯 | 任意 | ナビ上の細い帯。常設すると効果減 |
| スクロール誘導 | 任意 | 次セクションをFV下端に見切れさせるのが最も自然 |
| SNSリンク | 任意 | Instagramが実質作品集の業種のみFVに。他はフッター |
| 検索窓・予約フォーム | 条件付き | 宿泊・不動産・求人ではフォーム自体が主CTA |

## 3. 日本語キャッチコピーの型（10種）

| 型 | 構造 | 例 | 向く場面 |
|----|------|----|---------| 
| ターゲット指名型 | 「〜なあなたへ」と名指し | 夜、肩の痛みで目が覚める40代のあなたへ。 | 悩み解決型・LP |
| ベネフィット断言型 | 結果を数字・期限つきで断言 | 最短30分、その日のうちに直します。 | 修理・即日・SaaS |
| 数字・実績型 | 年数・件数・満足度が主役 | この街で創業47年。屋根の修理、12,000件。 | 老舗・施工・BtoB |
| 問いかけ型 | 現状に疑問を投げる | その保険料、あと20年払い続けますか？ | 見直し・乗り換え系 |
| 地域密着型 | 地域名＋業種＋強み | 三鷹駅徒歩3分。土日もやってる歯医者です。 | 店舗全般・ローカルSEO |
| 変化提示型 | Before→Afterの落差 | 請求書づくりの3時間が、3分になる。 | SaaS・教育・スクール |
| 逆説・意外性型 | 業界の常識を裏切る | 売り込みません。見積もりも急かしません。 | 差別化しにくい業種 |
| 情景・五感型 | 味・音・香りの場面描写 | 炭の爆ぜる音から、今夜の一杯が始まる。 | 飲食・宿・食品EC |
| 宣言・意志型 | 立場・約束を具体で宣言 | この町の空き家を、10年で100軒よみがえらせる。 | コーポレート・採用 |
| オファー直球型 | 価格・特典を最大サイズで | 初回体験60分 3,300円。今月は残り8枠。 | LP・キャンペーン |

### アンチパターン：抽象量産フレーズ（使用禁止）

「想いをカタチに。」「未来を創造する」「あなたに寄り添う」「もっと自由に、もっと楽しく」「笑顔をお届けします」——**どの会社が言っても成立するコピーは情報量ゼロ**であり、3秒で「何の店か」に答えられない。

- **入れ替えテスト**: そのコピーを競合や別業種のサイトに載せて違和感がなければ捨てる。「うちしか言えないか？」で判定
- **直し方**: 抽象語を「固有名詞・数字・場面」に1つずつ置換する。例：「想いをカタチに」→「手描きの図面1枚から、注文家具をつくって38年」
- 特に「宣言・意志型」はこのアンチパターンに転落しやすい。固有名詞・数字・期限のどれかを必ず入れる

## 4. LP（縦長訴求）とコーポレートFVの違い

LPのFVは「1つのオファーに向けた説得の開始点」、コーポレートのFVは「信頼と全体像の入口」。役割が別物なので、レイアウト・要素・コピーの選び方が変わる。

| 観点 | LP | コーポレート |
|------|----|------------| 
| ゴール | 単一CV（購入・予約・資料請求）へ直行 | 「誰の何の会社か」を伝え、複数目的へ振り分け |
| ナビ | 消すか最小化（離脱経路を断つ）。ロゴのみ残す | 必須。事業・会社情報・採用・お知らせ・問い合わせ |
| コピー | ベネフィット＋オファー＋緊急性（指名型・直球型・問いかけ型） | 自己紹介と信頼（宣言型・実績型・地域密着型） |
| CTA | FV内必須。1種類を繰り返す。マイクロコピー添え | 控えめ可。採用と事業など複数導線もあり |
| 要素 | 実績バッジ・限定数・締切・価格を前倒し。広告とメッセージ一致 | お知らせ更新枠・取引先ロゴなど「生きている会社」の証拠 |
| レイアウト | センター積み／左テキスト×右ビジュアル／オファー直球 | 全面写真／シネマティック／タイポ主役など世界観重視が許される |
| KPI | CVR・FV直帰率・スクロール到達率 | 回遊率・指名検索・下層遷移 |

**典型的な取り違え**: コーポレートにLPの煽り（今すぐ！残りわずか！）を持ち込む／LPで世界観だけ見せてオファー不在にする。
